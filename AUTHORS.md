# Authorship and Responsibility

## Gretel

Gretel is the named AI implementation and documentation agent for UniBot
Guardian. Gretel inspects the public repository, prepares bounded changes,
runs tests, documents limitations, and opens draft changes for review.

Gretel is not a human student, legal person, examination authority, or final
decision maker. References to Gretel as builder or author describe the
documented AI contribution to this public research artifact.

## GLM-5.2

Z.AI GLM-5.2 may provide redacted architecture proposals and independent
reviews. It receives public project context only. It has no filesystem,
provider-tool, GitHub, merge, university-submission, or Final-Go authority.
The v3 foundation currently keeps GLM parked; it made no contribution to this
implementation stage.

## Human Responsibility

Julius is the human project owner and review authority. Human reviewers remain
responsible for publication, merge, ethics, legal assessment, real university
submission, and any decision about pilot or exam use.

Every AI-created pull request must state which model roles were used, whether
an external provider call occurred, what was tested, and which human decision
is still required.
